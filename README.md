# Description
This project is the backend service for implementing these domain features in Temuka Platforms:
1. User Authentication 
2. User Profile Management
3. Feed Timeline
4. Community Management
5. Post Management
6. University Reviews
7. Major Reviews
8. Content Moderation
9. Cloud Storage Integration
10. Real Time Messaging
11. Notification

# Tech Stack
- Golang (Programming language)
- gRPC (Communication protocol with other services)
- PostgreSQL (Relational database)
- AWS S3 (Cloud file storage)
- Websocket (Bidirectional communication protocol)
- Redis (Non-relational and in memory database)
